## Getting Started

### Setup

* Install Expo CLI 
```sh
npm install --global expo-cli
```

* Clone Repository
```sh
$ git clone https://github.com/byeongjae-kang/react-native-practice.git
$ cd react-native-practice
```

* Install Dependencies
```sh
npm install
```

* Run Expo
```sh
npm start
```
